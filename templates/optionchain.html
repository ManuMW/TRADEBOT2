This endpoint fetches Historical Open Interest (OI) data for live F&O contracts. You need:

  • Try using current month's option/future contracts
  • Example: NIFTY current month futures or weekly options

üìã How to Find Symbol Tokens

To get valid symbol tokens for F&O contracts:

  1. Download the instrument list from Angel One: OpenAPI Scrip Master
  2. Search for your desired contract (e.g., "NIFTY", "BANKNIFTY")
  3. Filter by exchange: "NFO" and instrument type: "OPTIDX" or "FUTIDX"
  4. Copy the "token" value from the JSON
  5. Make sure the expiry date hasn't passed

Example: For NIFTY current month futures, look for entries with: "name": "NIFTY", "exch_seg": "NFO", "instrumenttype": "FUTIDX"
